Mongo Scraper will scrape spoofee.com website for special discount deals. The date that is retrieved will be latest deals and includes a link to the deal. Cheerio grabs the site content and Mongoose is used to save it to your MongoDB database. Users can select and save any deals they wish to have saved for later.

Some technologies included express, mongoose, body-parser, and cheerio.

Totally Trivial Trivia is a game that shows only one question until the player answers it or their time runs out. If the player selects the correct answer, it will show a screen congratulating them for choosing the right option. If the player runs out of time, it will tell the player that time's up and display the correct answer. If the player chooses the wrong answer, the player will be told they selected the wrong option and then display the correct answer. The final screen will show the number of correct answers, incorrect answers, and an option to restart the game (without reloading the page).

Some technologies included JavaScript for the logic and jQuery to manipulate HTML.
